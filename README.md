# PhonePe-End-to-End-Data-Analysis-Project
# 📱 PhonePe End-to-End Data Analysis Dashboard

An end-to-end data analytics project built using **Power BI** to analyze PhonePe transaction data. This dashboard provides interactive insights into transaction trends, payment status, service performance, and failed payment analysis across multiple financial services.

---

## 📌 Project Overview

This project analyzes PhonePe transaction data to monitor the performance of various digital payment services, including Insurance, Loans, Money Transfers, and Recharge & Bills. The dashboard enables users to explore transaction metrics through interactive visualizations and filters, helping identify trends, customer behavior, and operational issues.

---

## 🛠️ Tech Stack

- **Power BI** – Dashboard Development & Data Visualization
- **Microsoft Excel / CSV** – Data Source
- **Power Query** – Data Cleaning & Transformation
- **DAX** – Calculated Measures & KPIs

## 🔄 Project Workflow

1. Import transaction dataset into Power BI.
2. Clean and transform the data using Power Query.
3. Create data model and relationships.
4. Build DAX measures and KPIs.
5. Design interactive dashboards.
6. Add slicers and navigation buttons.
7. Publish and share the dashboard.
